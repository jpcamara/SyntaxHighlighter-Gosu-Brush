Overview

This brush allows you to add Gosu syntax highlighting to a page, when using the Syntax Highlighter by Alex Gorbatchev.

Syntax

It lets you change this:

classpath ".."
			
uses java.lang.Integer
			
var list = {1, 2, 3}
var map = {"a" -> "b"}
			
class MyClass implements List<Integer> {
	delegate _ints represents List<Integer> = {}
	var _data : String as Data
	var _lists : List<String> as Lists
}
			
var mine = new MyClass() {
	:Data = "some data",
	:Lists = {"list", "of", "data"}
}
mine.add(1)

Into this:

					classpath ".."

					uses java.lang.Integer

					var list = {1, 2, 3}
					var map = {"a" -> "b"}

					class MyClass implements List<Integer> {
						delegate _ints represents List<Integer> = {}
						var _data : String as Data
						var _lists : List<String> as Lists
					}

					var mine = new MyClass() {
						:Data = "some data",
						:Lists = {"list", "of", "data"}
					}
					mine.add(1)
Installation

Installation of SyntaxHighlighter is described here. To add in the Gosu highlighter, simply download this script and then include it after you include the syntax highlighter core files. for instance:

				<link href="your/path/to/styles/shCore.css" rel="stylesheet" type="text/css" />
				<link href="your/path/to/styles/shThemeDefault.css" rel="stylesheet" type="text/css" />
				<script type="text/javascript" src="your/path/to/scripts/shCore.js"></script>
				<script type="text/javascript" src="your/path/to/shBrushGosu.js"></script>
				<script type="text/javascript">
					SyntaxHighlighter.all();
				</script>
